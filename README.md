# static_string_calculation
## 本项目的是实现一个解析一段静态字符串的工具,实现思路来自于ts转js的编译流程
### 已完成:
#### 1.加减乘除四则运算
       1+2*3/4
#### 2.括号的解析
       1+(2-3)*4
#### 3.支持负数
       1+-2-3*4/-5
#### 4.支持变量
       {a:1,b:a+2} 1+2-a*b/4
#### 5.语法错误提醒
       1+2--3*4 ERROR:1+2[__-__]-3*4 
#### 6.函数解析
       1+2-min(1,2,3,4,5,6)
